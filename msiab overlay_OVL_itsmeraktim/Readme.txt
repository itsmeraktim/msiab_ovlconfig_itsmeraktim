MSIAB Minimal Lenovo OSD Overlay
Developed By_itsmeraktim.


A minimal MSI Afterburner (RTSS) OSD configuration built for Lenovo systems.
Clean layout. Essential metrics only. No clutter.

Requirements

MSI Afterburner (latest version recommended)

RivaTuner Statistics Server (RTSS)

Windows 10 / 11

Make sure RTSS is installed together with MSI Afterburner.

Installation Guide
Step 1 – Install Required Software

Download and install:

MSI Afterburner

RivaTuner Statistics Server (comes bundled)

Restart your PC after installation (recommended).

Step 2 – Import the Overlay File

Open MSI Afterburner

Go to Settings (gear icon)

Open the Monitoring tab

Enable the hardware metrics you want visible (CPU temp, GPU temp, usage, VRAM, FPS, etc.)

Tick "Show in On-Screen Display" for each selected metric

If your file is an RTSS layout preset:

Open RTSS

Load the provided overlay/profile file in the Overlay Editor

Apply and save

Step 3 – Enable OSD Hotkey

In MSI Afterburner Settings

Go to the On-Screen Display tab

Set a toggle hotkey (example: F10)

Apply changes

Recommended Metrics (Minimal Setup)

CPU Temperature

CPU Usage

GPU Temperature

GPU Usage

VRAM Usage

FPS

Frametime

Keep it simple. More metrics = more visual noise.

Why This Overlay?

Some Lenovo systems are reported to have thermal or motherboard-related concerns.
This layout helps you monitor critical performance and temperature metrics at a glance without cluttering your screen.

Clean. Functional. Efficient.

Notes

Works best for gaming and stress testing

Adjust font size and position inside RTSS if needed

Make sure RTSS is running in the background

If anything doesn’t show:

Check that RTSS is running

Make sure “Show in On-Screen Display” is enabled

Restart both applications

That’s it.
Simple setup. Clean monitoring.
